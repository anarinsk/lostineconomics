Adding bigfoot.js to Hugo

Junsok Huhh

2017/11/01

BIGFOOT!

웹서핑을 하다가 이 녀석을 보고야 말았다, bigfoot.js. 웹 문서가 고도화된 지금 인터넷으로 노출하는 문서에서 표현의 방법으로 답답함을 느끼는 일은 거의 없을 터다. 표, 수식, 그림 등을 동원해 소통할 때 pdf 등의 프린트 기반 문서들과 비교해 보다 기발하고 뛰어난 사례들도 많다. 당장 이 문서가 작성되고 있는 markdown이 그 대표 사례다.

어쨌든 왠만한 것들이 쉽게 된다는 걸 알게 되면, 작은 차이들이 눈에 들어오기 마련이다. 각주 처리만 좀 더 ‘멋지게’ 되면 좋겠는데, 싶었다. 마치 등이 살짝 가려운 데 긁지 못하는 격이랄까. 이때 bigfoot.js를 보게된 것이다! 각주를 누르면 버블로 해당 텍스트가 그 자리에 등장한다. 이거야말로 인터랙티브 문서가 지닌 장점이 아닌가!

그래서 바로 검색 들어 갔다. 우선, 필요한 요소를 챙겨보자.

  1. bigfoot.js를 구현하는 자바 스크립트
  2. 해당 자바스크립트를 이 공간이 사용하고 있는 Hugo framework에 배치하기

이 두가지만 얼추 해결되면 될 듯 싶었다. 어제 밤에 딱 10분 검색해서 찾아냈다.1

How to install2

원래 bigfoot.js는 ’점 세개’가 나오는 형태여서 숫자로 표기되는 전통적인 각주와는 어울리지 않는다. bigfoot.js는 보다 간략한 형태의 barefoot.min.js도 제공한다. 이 녀석을 붙여보기로 하자.

  1. barefoot github에서 barefoot.min.js, barefoot.min.css를 다운 받는다.
  2. barefoot.min.js을 static/js에 넣어준다. 혹시 static 디렉토리 안에 js가 없다면 만들면 된다.
  3. barefoot.min.css을 static/css에 넣어준다.
  4. themes/[YOUR_THEME_NAME]/layouts/partials/footer.html을 열어서 아래 코드를 적절한 위치에 삽입한다.
<link rel="stylesheet" href="/css/barefoot.min.css">

<script src="/js/barefoot.min.js"></script>
<script>
  lf = new BareFoot();
  lf.init();
</script>

How to use

그냥 markdown에 각주 쓰듯이 쓰면 되겠다.

This is barefoot![^baref]

[^baref]: 이것은 베어풋으로 붙인 각주입니다!
